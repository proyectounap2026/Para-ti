# Para ti

Una página web interactiva y romántica con efecto cyberpunk para hacer una invitación especial.

## ✨ Características

- 🔐 **Página de bloqueo**: Verificación de nombre para acceder al contenido
- 💝 **Invitación personalizada**: Muestra los detalles de la cita
- 💫 **Efectos visuales**: 
  - Animación Matrix rain de fondo
  - Código binario cayendo
  - Partículas de corazones flotantes
  - Efectos de shake y fade-in
- 🎨 **Diseño cyberpunk**: Colores rojo oscuro y tema futurista
- 📱 **Responsive**: Se adapta a cualquier tamaño de pantalla

## 🚀 Cómo usar

1. Clona el repositorio
2. Asegúrate de tener la carpeta `assets/` con el archivo `garu-pucca.gif`
3. Abre `index.html` en tu navegador
4. Escribe "Andres" para desbloquear la invitación

## 📋 Pasos

1. **Página 1**: Escribe tu nombre para desbloquear
2. **Página 2**: Confirma si aceptas la invitación
3. **Página 3**: Mensaje final con detalles de la cita

## 🎨 Personalización

Puedes cambiar los siguientes valores en el archivo `index.html`:

- **Nombre de desbloqueo**: Cambia `const UNLOCK_NAME = 'Andres';` en el script
- **Detalles de la cita**: Modifica la fecha, hora y ubicación en la sección de invitación
- **Colores**: Ajusta las variables CSS en el `:root`
- **GIF final**: Reemplaza `assets/garu-pucca.gif` con tu imagen

## 📁 Estructura

```
Para-ti/
├── index.html
├── assets/
│   └── garu-pucca.gif
└── README.md
```

## 🌐 Publicar con GitHub Pages

1. Ve a los settings del repositorio
2. Busca la sección "GitHub Pages"
3. Selecciona "Deploy from a branch"
4. Elige la rama `main` y la carpeta `/root`
5. ¡Listo! Tu página estará disponible en `https://tu-usuario.github.io/Para-ti/`

## 🎵 Sonido

Puedes agregar sonido de fondo modificando el script para incluir un elemento `<audio>`.
